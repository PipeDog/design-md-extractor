# Chrome Web Store 上架文案

本文档汇总 `DESIGN.md Extractor` 在 Chrome Web Store 上架时可直接复用的文案。

## 产品定位

- **Single purpose：** Analyze the current webpage UI and export a reusable DESIGN.md.
- **推荐分类：** Developer Tools
- **推荐默认语言：** English

## Store 名称

`DESIGN.md Extractor`

## 一句话描述

`Analyze the current webpage UI and export a reusable DESIGN.md in one click.`

## Short description

`Analyze any webpage UI and export a reusable DESIGN.md with local mode or AI mode.`

## Detailed description

```text
DESIGN.md Extractor helps you turn the current webpage into a reusable DESIGN.md document.

Use it to inspect page structure, UI patterns, design tokens, and visual signals, then export a structured Markdown file you can reuse in design analysis, prompt engineering, or UI system documentation workflows.

Key features:
- Analyze the current webpage and export a DESIGN.md file
- Use Local Mode for a fast draft without remote model calls
- Use AI Mode for stronger structure, higher-quality wording, and a more reusable result
- Configure your own model endpoint, API key, protocol, and token budget
- Save settings locally and download the final Markdown directly

The extension works only on normal http/https webpages and runs when you explicitly trigger it from the popup.

If AI Mode is enabled, the extension sends a structured page summary to the model endpoint you configure.
```
